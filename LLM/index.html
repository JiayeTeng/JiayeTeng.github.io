课程主要内容

课程目标

课程将介绍大模型的基础知识，包括它们的工作原理和发展历程，以及如何根据具体需求选择合适的工具。学生将通过分析大模型在自然语言处理、计算机视觉和推荐系统等领域的应用案例，了解这些工具如何被用来解决实际问题。课程还包括实践项目，让学生有机会亲自操作大模型工具，将理论知识转化为实际操作技能。

前置知识

  • 软件

    本课程主要使用 Python 作为编程语言。虽然课程中大部分代码由大模型直接生成，但读懂 Python 代码、理解代码的运行逻辑仍然可以提升同学们的课程理解。

  • 数学知识

    本课程在最后几周将会使用到线性代数（特别是矩阵运算）的相关知识。但课程考核中并不包含相关内容。掌握相关知识能够加强对本课程后半部分关于模型的理解。

  • 机器学习

    本课程在最后几周将会使用到机器学习（特别是 transformer）的相关知识。但课程考核中并不包含相关内容。
